# Verity
**A Verified Review Platform for Student Services**

## Overview
[cite_start]Verity is a prototype web application that allows students to share and view verified reviews of campus-based services and small businesses[cite: 7]. [cite_start]The project focuses on establishing trustworthiness, authenticity, and usability within a controlled, university environment[cite: 8, 9].

## Core Features
* [cite_start]**Service Directory**: Search and browse student-led services[cite: 13].
* [cite_start]**Verified Profiles**: View professional details, ratings, and visual trust cues[cite: 14].
* [cite_start]**Structured Reviews**: Submit fact-based, attribute-driven feedback to mitigate social bias[cite: 15].

## Technical Stack
* **Frontend**: React.js.
* **Backend**: Supabase (PostgreSQL).
* [cite_start]**Security**: Access restricted via university email domains[cite: 78].

## Research Focus
1. [cite_start]**Verification (RQ1)**: Balancing security with usability[cite: 42].
2. [cite_start]**Visual Trust (RQ2)**: Using UI patterns (Blue palette, badges) to mitigate skepticism[cite: 44, 49].
3. [cite_start]**Data Structure (RQ3)**: Improving comparability through structured review data[cite: 45].
