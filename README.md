# Verity
**A Verified Review Platform for Student Services**

## Overview
Verity is a prototype web application that allows students to share and view verified reviews of campus-based services and small businesses. The project focuses on establishing trustworthiness, authenticity, and usability within a controlled, university environment.

## Core Features
* **Service Directory**: Search and browse student-led services.
* **Verified Profiles**: View professional details, ratings, and visual trust cues.
* **Structured Reviews**: Submit fact-based, attribute-driven feedback to mitigate social bias.

## Technical Stack
* **Frontend**: React.js
* **Backend**: Supabase (PostgreSQL)
* **Security**: Access restricted via university email domains.

## Research Focus
1. **Verification**: Balancing security with usability.
2. **Visual Trust**: Using UI patterns (Blue palette, badges) to mitigate skepticism.
3. **Data Structure**: Improving comparability through structured review data.
